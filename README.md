# Почесні Газди — Асоціація

Вебсайт асоціації "Почесні Газди" — React + TypeScript + Tailwind CSS + shadcn/ui

## Запуск проєкту

```bash
npm install
npm run dev
```

Або для збірки:
```bash
npm run build
npm run preview
```

## Структура

```
src/
  sections/       # Секції сайту
    Header.tsx
    Hero.tsx
    About.tsx
    FocusAreas.tsx
    Projects.tsx
    Team.tsx
    News.tsx
    Partners.tsx
    Donate.tsx
    Contact.tsx
    Footer.tsx
  components/ui/  # shadcn/ui компоненти
  lib/utils.ts    # Утиліти
  App.tsx         # Головний компонент
  main.tsx        # Точка входу
  index.css       # Глобальні стилі + CSS змінні
```
